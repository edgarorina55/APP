📦 Installation & Setup
Clone the repository:
    git clone https://github.com/edgarorina55/APP.git
    cd apps

Install dependencies for any app:
    cd api/my-app
    npm install 

    or for Python:
        pip install -r requirements.txt 

▶️ Running an App
Node.js
    cd api/my-node-app
    npm start 

Python 
    cd api/my-fastapi-app
    uvicorn main:app --reload

Database scripts 
    cd db 
    mysql -u root -p < schema.sql
